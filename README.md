# catshow--ts

学习 TypeScript 的 catshow 项目

## 基础环境搭建

### TypeScript

`Bash`:

```bash
npm i -g typescript
tsc --init
tsc -w
```

### Git

`Bash`:

```bash
git init
git add -A
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/cicadasinging/catshow--ts
git push -f origin main
```
