# catlist--ts

学习 TypeScript 的 catlist 示例

## 开发环境搭建

### TypeScript

`Bash`:

```bash
npm i -g typescript
tsc --init
tsc -w
```

### Git

`Bash`:

```bash
git init
git add -A
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:cicadasinging/catlist--ts.git
git push -f origin main
```
